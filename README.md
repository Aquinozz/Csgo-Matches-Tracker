# Csgo-Matches-Tracker
E-sports Hub (CS2)  Aplicação para monitoramento de partidas competitivas em tempo real.  Destaques:  Back-end: Python &amp; Flask com integração da API PandaScore.  Front-end: Interface moderna em Dark Mode com CSS puro e Jinja2.  UX: Sistema de tratamento de falhas em imagens e layout 100% responsivo.  Python | Flask | REST API | CSS3
